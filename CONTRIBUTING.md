# Contributing

Thanks for contributing!

## What to contribute

- Improvements to `SKILL.md` writing workflow and checklists
- More practical examples for different occasions (in `references/`)
- Fixes and enhancements to `scripts/pingze_check.py`

## Guidelines

- Keep language concise and actionable.
- Avoid long verbatim quotes from copyrighted sources.
- For “web research” guidance, prefer describing workflows over linking to a single source.
- For changes that affect usage, update `README.md` accordingly.
